# AXIOM Protocol Documentation

**AI-Powered Real World Asset Protocol on Solana**

AXIOM ($AXM) deploys intelligent AI agents that autonomously manage, optimize, and govern tokenized real-world assets on Solana — from treasury bonds to real estate, commodities to private credit.
