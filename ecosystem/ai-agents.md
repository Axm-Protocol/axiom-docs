# AI Agent Marketplace

The AXIOM AI Agent Marketplace is a decentralized hub where users can deploy, customize, and share autonomous portfolio management agents.

## Available Agent Types

| Agent | Strategy | Risk Level |
|-------|----------|-----------|
| **T-Bill Yield Agent** | Optimizes allocation across tokenized US Treasury bills for maximum risk-adjusted yield | Low |
| **DeFi Yield Agent** | Routes idle capital to top Solana DeFi protocols — stablecoin pools, LSTs, lending | Medium |
| **Multi-Asset RWA Agent** | Diversified portfolio across real estate, commodities, and fixed income | Medium |
| **Alpha Seeker Agent** | Aggressive strategy combining RWA yields with DeFi leverage and arbitrage | High |

## How to Deploy an Agent

1. Browse available agents on the AXIOM marketplace
2. Select an agent and review its strategy, historical performance, and risk profile
3. Configure parameters (risk tolerance, asset preferences, rebalancing frequency)
4. Deposit assets into the agent's managed vault
5. Monitor performance on the dashboard

## Building Custom Agents

Developers can build and publish custom AI agents on the marketplace. See the [Building AI Agents](../developers/building-agents.md) developer guide for technical details.

## Agent Revenue

- Agent creators earn a percentage of management fees from users
- Top-performing agents are featured in the marketplace
- Revenue is distributed in $AXM tokens
