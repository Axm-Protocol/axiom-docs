## Security Measures

- **Multi-sig authority:** All admin functions require multi-signature approval
- **Timelock controls:** Governance actions have a 48-hour timelock before execution
- **Emergency pause:** Admin can pause critical functions in case of exploit detection
- **Oracle redundancy:** Multiple price feeds to prevent oracle manipulation
- **Rate limiting:** On-chain rate limits to prevent flash loan attacks

## Bug Bounty

> If you find a security vulnerability, please report it responsibly to **security@axmprotocol.io**. We take all reports seriously and will respond within 24 hours.

## Best Practices for Users

- Always verify you're on the official website (axmprotocol.io)
- Never share your seed phrase or private keys
- Use a hardware wallet for large holdings
- Verify transaction details before signing
- Be cautious of phishing links in Discord/Telegram
